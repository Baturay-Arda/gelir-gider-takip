# Gelir Gider Takip Uygulaması
Python + CustomTkinter ile geliştirilmiş masaüstü finans takip uygulaması.

## Özellikler
- Gelir / Gider ekleme
- Kategori sistemi
- Tarih takibi
- Kayıt silme ve güncelleme
- Grafik (matplotlib)
- JSON ile kalıcı veri

## Kullanılan Teknolojiler
- Python
- CustomTkinter
- Matplotlib
- JSON

## Çalıştırma
```bash
python gelir_gider.py
